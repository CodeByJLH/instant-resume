# Resume Builder Pro

纯前端的在线简历编辑器 — 无需框架、无需构建工具，打开 `index.html` 即可使用。

## 快速开始

双击 `index.html` 或在浏览器中直接打开即可。所有依赖（html2canvas、jsPDF）通过 CDN 加载，无需 `npm install`。

## 功能概览

- **双语模板** — 中英文一键切换
- **内容可编辑** — 所有文字点击即改，支持动态增删条目
- **头像上传** — 点击上传照片，支持拖拽/缩放调整，圆形/圆角/方形裁剪
- **排版调节** — 行高、字间距、内边距、字号、字体均可精细控制
- **纸张与版式** — 竖版/横版切换，支持 A4/Letter 等标准尺寸及自定义尺寸
- **主题色** — 8 种预设色板 + 自定义取色器
- **PDF 导出** — 一键下载高质量 PDF

## 技术栈

原生 HTML + CSS + JavaScript，无任何框架。CSS 自定义属性驱动主题化，html2canvas + jsPDF 实现 PDF 导出。
